# Mayuri Satpute - Premium Corporate Portfolio Website

A modern, premium, and fully responsive personal portfolio website for Mayuri Satpute, an MBA graduate specializing in Human Resources and Marketing. Designed to attract high-level recruiters for ₹10–20 LPA positions.

## Features

- **Premium Corporate Design**: Navy blue, gold accents, and sophisticated typography
- **Professional Color Palette**: Navy blue (#1a365d), dark gray (#2d3748), gold (#d4af37)
- **Advanced Animations**: Smooth scrolling, staggered fade-ins, hover effects, and floating elements
- **Mobile-First Responsive**: Optimized for all devices with Bootstrap 5
- **SEO Optimized**: Meta tags and semantic HTML for better search visibility
- **Interactive Elements**: Enhanced hover effects, smooth transitions, and micro-interactions

## Sections

1. **Hero Section** - Profile image, name, tagline, and CTA buttons with premium styling
2. **About Me** - Professional summary with image and key strengths
3. **Skills** - Icon-enhanced skill cards in premium layout
4. **Projects** - Detailed project showcases with badges and icons
5. **Resume** - Premium download button
6. **Contact** - Professional contact cards with social links
7. **Footer** - Enhanced footer with social icons and branding

## Tech Stack

- HTML5
- CSS3 (Custom properties, gradients, animations)
- JavaScript (ES6+)
- Bootstrap 5
- Google Fonts (Montserrat, Open Sans)
- Font Awesome Icons

## Required Assets

Add the following images to the `assets/` folder:

1. `profile.jpg` - Professional headshot (150x150px, circular crop recommended)
2. `about.jpg` - Professional photo for about section (any aspect ratio, will be responsive)
3. `resume.pdf` - Your resume PDF file

## Customization

- Update personal information in `index.html`
- Modify colors in `css/styles.css` (CSS custom properties)
- Adjust animations in `js/script.js`
- Replace placeholder images with your professional photos
